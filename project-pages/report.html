Accident Report

What is a Accident Report?

An Accident Report is a formal document that presents information that has been gathered from a technical or significant accident. It presents techinical information in an organized document that is easy to follow (Headings, Chapters, photos, ect.). Also, with presenting facts, it also provides an analysis on how the accident occurred, and suggestions on how to prevent a similar accident from happening again.

How did I complete this project

For my group's accident report, we decided to report on an airline accident that involved a collision between a grounded aircraft and a fuel tanker. We based our report on an existing accident report on the incident, where we presented the same findings and main ideas in the original report in our own words. Each section in the original document was divided among members and presented in the document in 4 main sections, which were background information, accident analysis, safety actions, and conclusion.

Formal document writing and research were essential skills that I learned when working on this project.

What did I learn?

Formal Document Design

This includes a table of contents that displays each section in the document, clear headings, labeled visual media, and citations.

Report Research Effectivly

Gather all relevant information and media and present findings in an summarized and organized manner.

Formal Writing

Presenting techinical information and a clear and concise manner while maintaining a unbiased and objective tone.
